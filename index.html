 สวัสดีครับ

 ภาษา python เป็นภาษาโปรแกรมชนิด interpreted มีคุณสมบติรองรับการโปรแกรมแบบ object-oriented programming สร้างโดย Guido van Rossum ระหว่างปี 1985-1990 เนื้อหาที่เรียนจะใช้ python3 ที่ถูกปล่อยออกมาใช้งานตั้งแต่ปี 2008 เนื้อหาในหลักสูตรต้อง การให้ผู้เรียนสามารถเรียนรู้การใช้งานภาษา python เพื่อนำไปประยุกต์การใช้งานต่อไป
